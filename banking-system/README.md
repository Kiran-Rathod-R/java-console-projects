# Banking System – Java Console Project

This project simulates a **basic banking system** using Java where users can perform simple banking operations.

---

## Features

* Create account
* Deposit money
* Withdraw money
* Check balance

---

## Concepts Used

* Java classes and objects
* Methods
* Conditional statements
* Scanner for user input

---

## Project Structure

banking-system
│
├── BankSystem.java
└── README.md

---

## How to Run

Compile:

javac BankSystem.java

Run:

java BankSystem
