# Complex Number Calculator – Java Console Project

This project is a **Java console application that performs arithmetic operations on complex numbers**.

The program allows the user to enter two complex numbers and perform mathematical operations such as addition, subtraction, multiplication, and division.

---

## Features

* Input two complex numbers
* Addition of complex numbers
* Subtraction of complex numbers
* Multiplication of complex numbers
* Division of complex numbers
* Display complex numbers

---

## Concepts Used

This project demonstrates important **Java programming concepts**:

* Classes and Objects
* Constructors
* Methods
* Arithmetic operations
* Menu-driven programs
* User input using Scanner
