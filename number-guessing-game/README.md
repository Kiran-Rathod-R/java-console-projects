# Number Guessing Game – Java Console Project

This is a simple **number guessing game** written in Java where the player must guess a randomly generated number.

---

## Features

* Random number generation
* User input for guessing
* Hints if guess is too high or too low
* Limited attempts

---

## Concepts Used

* Random class
* Loops
* Conditional statements
* Scanner class for input

---

## Project Structure

number-guessing-game
│
├── NumberGuessingGame.java
└── README.md

---

## How to Run

Compile:

javac NumberGuessingGame.java

Run:

java NumberGuessingGame

---

## Gameplay

* The system generates a random number.
* The user keeps guessing until the correct number is found.
* The program provides hints.
