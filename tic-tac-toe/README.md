# Tic Tac Toe – Java Console Game

A simple **Tic Tac Toe game** built using Java that runs in the command line.
The game allows two players to play turn-by-turn and determines the winner based on the board state.

---

## 🎮 Features

* Two player gameplay (Player X vs Player O)
* Console based game board
* Player turn switching
* Winner detection
* Draw detection when board is full
* Input validation

---

## 🧠 Concepts Used

This project demonstrates several core Java programming concepts:

* Arrays
* Loops
* Conditional statements
* Methods
* User input using Scanner
* Basic game logic implementation

## ▶️ How to Run

1. Compile the program

```
javac TicTacToe.java
```

2. Run the program

```
java TicTacToe
```

## 🖥 Example Gameplay

```
 X | O | X
-----------
 O | X | O
-----------
 X |   | O
```

Player **X wins the game**.
