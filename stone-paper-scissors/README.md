# Stone Paper Scissors – Java Console Game

This project is a **Stone Paper Scissors game implemented in Java** where a user plays against the computer.

The computer randomly selects a move, and the program determines the winner.

---

## Features

* Player vs Computer gameplay
* Random computer choice
* Game result detection
* Console-based interface

---

## Concepts Used

* Random number generation
* Conditional statements
* User input using Scanner
* Basic game logic

---

## Project Structure

stone-paper-scissors

└── README.md

---

## How to Run

Compile the program

javac StonePaperScissor.java

Run the program

java StonePaperScissor

---

## Gameplay

1. User selects an option:

   * 1 → Stone
   * 2 → Paper
   * 3 → Scissors

2. Computer randomly selects a move.

3. The program compares both moves and declares the winner.
