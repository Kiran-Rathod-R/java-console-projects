# Java Console Projects 🚀

This repository contains several **Java console-based mini projects** created to practice core programming concepts and improve problem-solving skills.

These projects demonstrate fundamental Java features such as loops, conditional statements, user input handling, and object-oriented programming.

---

## 📂 Projects Included

### 1. To-Do Task Manager
A simple console-based task manager that allows users to insert, remove, and display tasks.

### 2. Stone Paper Scissors Game
A command-line game where the user plays Stone-Paper-Scissors against the computer.

### 3. Percentage Calculator
A program that calculates the percentage based on marks entered for five subjects.

### 4. Complex Number Operations
A Java program that performs arithmetic operations on complex numbers.

---

## 🛠 Technologies Used

- Java
- Object Oriented Programming (OOP)
- Arrays
- Loops
- Conditional Statements
- Scanner Class for User Input

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/Kiran-Rathod-R/java-console-projects.git
```

### 2. Navigate to the repository folder

```
cd java-console-projects
```

### 3. Compile the Java file

```
javac FileName.java
```

### 4. Run the program

```
java FileName
```

Example:

```
javac complex_number.java
java complex_number
```
