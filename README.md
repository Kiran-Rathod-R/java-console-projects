# Java Console Projects

This repository contains several console based Java projects created to practice core programming concepts.

## Projects Included

1. Calculator
2. Tic Tac Toe Game
3. Number Guessing Game
4. Student Management System
5. Banking System

---

## Technologies Used

* Java
* Object Oriented Programming
* Arrays
* Loops
* Conditional Statements
* Scanner Class for User Input

---

## How to Run

1. Clone the repository

git clone https://github.com/Kiran-Rathod-R/java-console-projects.git

2. Navigate to project folder

3. Compile the program

javac FileName.java

4. Run the program

java FileName
