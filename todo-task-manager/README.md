# To-Do Task Manager – Java Console Application

This project is a **simple task management system built using Java** that allows users to manage daily tasks from the command line.

Users can add tasks, mark tasks as completed, and display the task list.

---

## Features

* Add new tasks
* Mark tasks as completed
* Display all tasks
* Menu driven program
* Simple console interface

---

## Concepts Used

This project demonstrates the following Java concepts:

* Classes and objects
* Arrays
* Loops
* Conditional statements
* Scanner class for user input
